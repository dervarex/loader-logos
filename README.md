# Mod Loader Logos

A centralized collection of Minecraft mod loader logos used by **Minified**.

This repository allows Minified and other applications to fetch official mod loader logos dynamically instead of bundling them with each release.

## Usage

Example:

```text
https://raw.githubusercontent.com/dervarex/loader-logos/main/fabric/fabric.png
```
